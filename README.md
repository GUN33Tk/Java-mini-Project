# Java-mini-project
The Advanced URL Safety Checker is a Java-based tool that analyzes URLs to detect potential threats. It uses VirusTotal for real-time scanning and applies heuristic checks like keywords, HTTPS, IP usage, and typosquatting to classify URLs as safe or suspicious.
